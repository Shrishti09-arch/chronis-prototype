# Chronis Behavioral Analytics Prototype

An interactive frontend platform built for the Chronis Internship Track Assessment. This application parses complex bio-telemetry trends, maps multi-period narrative timelines, surfaces anomalies, and explicitly accounts for tracking uncertainty with contextual data-integrity alerts.

## 🚀 Features Implemented
* **Interactive Dashboard:** Core correlation trend mapping utilizing `recharts` for tracking focus scores, sleep architecture degradation, and stress metrics.
* **Insight Explorer:** Deep-dive navigation system that pulls telemetry evidence and details missing data streams to ensure system transparency.
* **Narrative Timeline:** A historical multi-segment component tracking user behavioral evolution over macro phases with interactive status filtering.
* **Modern Architecture:** Configured natively under React 19, Vite, and Tailwind CSS v4.
